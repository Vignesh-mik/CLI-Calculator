# CLI Calculator

## Description
A simple command-line calculator that performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features
- Supports +, -, *, /
- User-friendly input
- Fast execution

## Tech Used
- C++

## How to Run
1. Compile the code:
   g++ calculator.cpp
2. Run:
   ./a.out

## Sample Output
Enter two numbers: 5 3  
1.Add 2.Subtract 3.Multiply 4.Divide  
Choice: 1  
Result: 8
